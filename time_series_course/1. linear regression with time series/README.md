# Linear Regression with Time Series

This module demonstrates how to apply linear regression techniques to time series data.

## Contents

- `exercise-linear-regression-with-time-series.ipynb` - Notebook exploring regression-based forecasting and model interpretation.

## Learning Goals

- Understand how linear regression can be used for time series prediction
- Explore time-based feature engineering
- Interpret regression coefficients in a temporal context

## Usage

Open the notebook in Jupyter and run the analysis cells to follow the workflow from data preparation through model evaluation.
