# Seasonality

This module covers seasonality detection and pattern analysis in time series.

## Contents

- `exercise-seasonality.ipynb` - Notebook demonstrating seasonal decomposition and interpretation.

## Learning Goals

- Identify seasonal cycles in time series data
- Apply decomposition techniques to separate trend and seasonality
- Interpret seasonal behavior for forecasting

## Usage

Open the notebook in Jupyter and run the analysis cells to explore seasonal patterns.
