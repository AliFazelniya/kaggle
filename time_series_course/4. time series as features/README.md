# Time Series as Features

This module explores how to treat time series data as feature sets for machine learning.

## Contents

- `exercise-time-series-as-features.ipynb` - Notebook on feature engineering and model building using time series-derived features.

## Learning Goals

- Generate time-based features for predictive models
- Understand how time series data can be represented as feature vectors
- Use machine learning models with engineered temporal features

## Usage

Open the notebook in Jupyter and execute the cells to follow the feature engineering workflow.
