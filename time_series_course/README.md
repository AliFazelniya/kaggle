# Time Series Course

This folder contains a set of time series learning exercises, divided into six focused modules. Each module is delivered as a Jupyter notebook and explores a different aspect of time series analysis and forecasting.

## Contents

- `1. linear regression with time series/` - Linear regression applied to time series data.
- `2. trend/` - Trend detection and modeling techniques.
- `3. seasonality/` - Seasonal pattern analysis and decomposition.
- `4. time series as features/` - Feature engineering strategies for time series models.
- `5. hybrid models/` - Combining models and hybrid forecasting approaches.
- `6. forecasting with machine learning/` - Machine learning methods for time series forecasting.

## Usage

1. Open the desired module notebook in Jupyter or VS Code.
2. Install or activate a Python environment with packages such as `pandas`, `numpy`, `matplotlib`, `seaborn`, and `scikit-learn`.
3. Run the notebook sequentially to follow the exercise flow and reproduce results.

## Purpose

This course-style folder is designed for learners who want to:

- Understand foundational time series concepts
- Practice trend and seasonality analysis
- Build forecasting models using regression and machine learning
- Explore feature-based and hybrid modeling workflows
