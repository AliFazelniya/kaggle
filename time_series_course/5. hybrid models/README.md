# Hybrid Models

This module introduces hybrid modeling approaches for time series forecasting.

## Contents

- `exercise-hybrid-models.ipynb` - Notebook covering hybrid combinations of statistical and machine learning techniques.

## Learning Goals

- Learn the concept of hybrid time series models
- Explore the benefits of combining multiple modeling approaches
- Evaluate hybrid forecasts against simpler baselines

## Usage

Open the notebook in Jupyter and run the analysis cells to explore hybrid forecasting strategies.
