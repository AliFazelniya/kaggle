# Forecasting with Machine Learning

This module explores machine learning techniques for time series forecasting.

## Contents

- `exercise-forecasting-with-machine-learning.ipynb` - Notebook demonstrating regression and ML-based forecasting methods.

## Learning Goals

- Apply machine learning algorithms to forecast time series
- Compare ML-based forecasts to traditional approaches
- Evaluate prediction accuracy for regression models over time

## Usage

Open the notebook in Jupyter and execute the cells to follow the machine learning forecasting workflow.
