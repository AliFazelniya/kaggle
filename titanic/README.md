# Titanic Survival Prediction

This folder contains a Titanic survival prediction project based on the classic Kaggle competition dataset.

## Contents

- `train.csv` - Training dataset with passenger features and survival labels.
- `test.csv` - Test dataset for model prediction and evaluation.
- `main.ipynb` - Notebook with data exploration, feature engineering, modeling, and evaluation.

## Project Overview

The notebook is designed to:

- Explore the Titanic passenger dataset and understand feature relationships
- Engineer features useful for survival prediction
- Build and evaluate classification models
- Visualize model performance and interpret results

## Usage

1. Open `main.ipynb` in Jupyter or VS Code.
2. Install required dependencies such as `pandas`, `numpy`, `matplotlib`, `seaborn`, and `scikit-learn`.
3. Execute the notebook to reproduce the analysis and model workflow.

## Notes

- This project is suitable for learning feature engineering and classification modeling.
- The dataset is structured for the Kaggle Titanic competition format.
