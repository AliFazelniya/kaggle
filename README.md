# Kaggle Project Collection

This repository contains a collection of Kaggle-based data science and machine learning projects focused on credit card fraud detection, Titanic survival prediction, and time series forecasting. Each folder contains notebooks, datasets, and supporting code for exploratory analysis, modeling, and visualization.

## Repository Structure

- `credit_card_fraud_detection/` - A fraud detection project with separate training and test datasets.
- `creditcardfraud/` - A second credit card fraud analysis project using the full credit card dataset.
- `time_series_course/` - A structured set of time series learning exercises with six focused modules.
- `titanic/` - A Titanic survival prediction project with training and test datasets.
- `utility/` - Reusable visualization utilities for plotting distributions, correlations, confusion matrices, and ROC curves.

## Getting Started

1. Open the repository in VS Code or JupyterLab.
2. Install Python dependencies for data science and machine learning, such as:
   - `pandas`
   - `numpy`
   - `matplotlib`
   - `seaborn`
   - `scikit-learn`

3. Launch the notebooks directly from the folder of interest.

## Notes

- This repository is primarily exploratory and educational in nature.
- The notebooks are intended for interactive analysis, model development, and visualization.
- Dataset files are stored locally in the project folders.

## Recommended Workflow

- Use the `utility/plot_utils.py` functions to standardize visualization across experiments.
- Start with `credit_card_fraud_detection/` or `titanic/` for classification practice.
- Use `time_series_course/` for time series concepts and forecasting exercises.
