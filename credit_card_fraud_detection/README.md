# Credit Card Fraud Detection

This folder contains a focused fraud detection project using transaction data, with separate training and test datasets and a Jupyter notebook for analysis.

## Contents

- `fraudTrain.csv` - Training dataset for fraud detection.
- `fraudTest.csv` - Test dataset for model evaluation.
- `main.ipynb` - Primary notebook containing data exploration, modeling, and validation steps.

## Project Overview

The notebook is designed to:

- Explore and understand the fraud dataset
- Clean and preprocess transaction features
- Build classification models to detect fraudulent transactions
- Evaluate model performance using confusion matrices, ROC curves, and other metrics

## Usage

1. Open `main.ipynb` in Jupyter or VS Code.
2. Ensure dependencies like `pandas`, `numpy`, `matplotlib`, `seaborn`, and `scikit-learn` are installed.
3. Execute the notebook cells sequentially to reproduce the analysis.

## Expected Outcome

The project demonstrates how to handle imbalanced classification, interpret model results, and visualize performance metrics for fraud detection scenarios.
